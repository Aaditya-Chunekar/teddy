1. - -Wall - Enables most common compiler warnings (e.g., unused variables, implicit function declarations)
    - -Wextra - Enables additional warnings beyond -Wall (e.g., missing field initializers, signed/unsigned comparisons)
    - -pedantic - Enforces strict ISO C standard compliance and warns about non-standard extensions
    - -std=c99 - Specifies the C99 standard for compilation (supports features like variable declarations mid-block, inline functions, etc.)
